README – Student Management System (CSE445 Project) 
Overview: 
This project is a simple CRUD-based Student Management System built for the CSE445: Web 
Application & Internet course. It demonstrates full-stack web development using HTML, CSS, 
JavaScript, PHP and MySQL. Students can add, view, edit and delete student records. 
Features: 
• Add new student 
• View student list 
• Update student information 
• Delete student 
• Login/Logout authentication system 
• Search/filter functionality 
• Error handling and feedback messages 
Technologies Used: 
1. Frontend: HTML, CSS, JavaScript 
2. Backend: PHP (Procedural) 
3. Database: MySQL (via XAMPP) 
4. Security: Prepared statements to prevent SQL injection
